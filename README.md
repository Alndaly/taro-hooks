## 特性

- 全面匹配`Taro API`.
- 结合`ahooks`扩展常用`hook`.
- 完整的类型定义文件
- 更易用的方式

## 安装

```bash
$ npm i taro-hooks --save
```

## 使用

```jsx
import { useEnv } from 'taro-hooks';
```

## 参与贡献

```bash
$ git clone git@github.com:innocces/taro-hooks.git
$ cd taro-hooks
$ yarn
$ yarn start
```

会自动打开浏览器 - [http://localhost:12345](http://localhost:12345)  
更多贡献详情见[官网文档]()
