# useFile

上传、下载文件

## 何时使用

当需要上传下载文件时

## API

```jsx | pure
const visible: boolean = useVisible();
```

## 返回值说明

| 返回值  | 说明                 | 类型      |
| ------- | -------------------- | --------- |
| visible | 当前应用是否处于后台 | `boolean` |
